# 🔐 Password Generator GUI (Python)

A modern and user-friendly password generator built using Python and Tkinter.  
This app allows users to generate secure passwords with customizable options.

---

## 🚀 Features

- 🔢 Adjustable password length (Slider)
- 🔤 Include letters, numbers, and symbols
- 🔐 Password strength indicator (Weak / Medium / Strong)
- 📋 Copy password to clipboard
- 🎨 Clean dark-themed UI
